Downloading Images

This page describes how to download images from a Symbiota portal.

It is incumbent upon data users to abide by usage rights when downloading images from Symbiota portals. Usage rights appear on collection profiles, occurrence records, and in each portal community’s respective Data Usage Guidelines (Sitemap > Image Library > Usage Policy and Copyright Information).
